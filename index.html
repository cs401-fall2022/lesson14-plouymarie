TypeScript 101

Intro

TypeScript is basically a layer on top of Javascript that enforces how types are assigned. Below we will go over some of the basics of TypeScript.

Types

By Inference

Typescript knows the Javascript language, therefore when declaring a variable, Typescript will generate the type for you.

let helloWorld = "Hello World"

The above example infers helloWorld as a type string. let helloWorld : string

Defining

You can also define types when design patterns make it difficult for types to be inferred.
For example, to create an object, you can write:

const user = {
  name = "Marie";
  age = 23;
}

where name is inferred as name : string, and age as age : number

To create this object's shape, you can use an interface declaration:

interface user = {
  name = string;
  age = number;
}

Since JavaScript supports classes and object-oriented programming, so does TypeScript. You can use an interface declaration with classes:

interface user {
  name : string;
  age : number;
}

class UserAccount {
  name : string;
  age : number;

  constructor ( name : string, age : number) {
   this.name = name;
   this.age = age;
  }
}

Composing Types

In TypeScript, we can create complex types by combining simple one. You can so so with Unions or Generics.

Unions

With a union, you can declare that a type could be one of many types

type MyBool = true | false;

The above example describes a boolean type as being either true or false.

Unions provide a way to handle different types as well:

function getFunction ( obj: string | string[]) {
  return obj.length;
}
Generics

Generics provide variables to types. A common example is an array. An array without generics could contain anything. An array with generics can describe the values that the array contains.

type StringArray = Array< string>;
type NumberArray = Array< number>;
type ObjectWithNameArray = Array<{ name: string }>;

Functions

Functions in Typescript are the basic building blocks. They can be declared in many different ways:

function helloWorld ( s: string ) => void) {
  fn("Hello World");
}

function printToCL ( s: string ) {
  console.log(s);
}

helloWorld(printToCL);
